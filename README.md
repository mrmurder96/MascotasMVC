## ❓ PREGUNTAS FRECUENTES

### P: ¿Qué es ARIA?
**R:** Sistema de atributos HTML que hace aplicaciones web accesibles para lectores de pantalla (para personas ciegas).

### P: ¿Debo hacer algo ahora?
**R:** Sí, lee QUICK_START.md y decide qué hacer según tu rol.

### P: ¿Cuánto tiempo tarda todo?
**R:** Fase 1 (hecho): 1 día. Fases 2-5: ~1 semana.

### P: ¿Tengo que instalar NVDA?
**R:** No obligatorio, pero muy recomendado para verificar.

### P: ¿Qué es WCAG 2.1?
**R:** Estándar internacional de accesibilidad web.

### P: ¿Esto me beneficia a mí?
**R:** Sí, navegación más clara y mejor UX para todos.
