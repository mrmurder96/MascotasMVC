## Cómo ejecutar
1. Clona el repo:
   ```bash
   git clone https://github.com/mrmurder96/MascotasMVC.git
   ```
2. Abre `Integrador.sln` en Visual Studio.
3. Configura la cadena de conexión (si aplica).
4. Ejecuta con F5.
