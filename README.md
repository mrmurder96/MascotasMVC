# Pura Compañía - Sistema de Adopción de Mascotas

Sistema web completo para la gestión de adopción de mascotas, desarrollado con ASP.NET MVC 5 y Entity Framework 6.

## Características

- Gestión Completa de Mascotas
- Sistema de Adopción
- Campañas con imágenes
- Panel de Administración
- Sistema de Roles y Permisos
- Carga de Imágenes con validaciones
- Autenticación Segura

## Tecnologías

- ASP.NET MVC 5 (.NET Framework 4.8)
- Entity Framework 6
- SQL Server
- Bootstrap 5
- jQuery 3.7
- Font Awesome 6

## Instalación

1. Clonar el repositorio: `git clone https://github.com/mrmurder96/MascotasMVC.git`
2. Restaurar la base de datos desde /Database/
3. Configurar cadena de conexión en Web.config
4. Restaurar paquetes NuGet
5. Compilar y ejecutar (F5)
